MASTER Looker Studio Codebase Manual

Calculated formulas & rules to build Table "2. Monthly Comparison (YoY 2025 vs 2026)"

Create these calculated fields in your Google Sheets data source configuration in Looker Studio (click "Add Field" in the bottom right of the schema mapping list):

2025 Total Formula 
SUM(CASE WHEN Year = 2025 THEN CAST(Total AS NUMBER) ELSE 0 END)
2026 Total Formula 
SUM(CASE WHEN Year = 2026 THEN CAST(Total AS NUMBER) ELSE 0 END)
% Change YoY Math Formula 
IFERROR(((2026 Total - 2025 Total) / NULLIF(2025 Total, 0)) * 100, CASE WHEN 2025 Total = 0 AND 2026 Total > 0 THEN 100 WHEN 2025 Total = 0 AND 2026 Total < 0 THEN -100 ELSE 0 END)
Trend Indicator Formula 
CASE WHEN % Change > 0 THEN "▲" WHEN % Change < 0 THEN "▼" ELSE "●" END
Month_Num Chronological Rank Formula 
EXTRACT(MONTH FROM Date)

Conditional Styling & Formats

  • Thousand Separator Formatting: For both 2025 Total and 2026 Total, choose "Numeric" format with thousand separators. Turn off unit symbols (Set secondary currency to "None").
  • Percentage Formatting: Choose % Change style representation as Numeric with suffix % and exactly 1 decimal digit precision.
  • Trend Unicode Conditional Color Configuration: Go to Style configuration panel › Conditional Formatting rules:
    • Rule 1: If "Trend" equals "▲", set Text color to Green (#10B981)
    • Rule 2: If "Trend" equals "▼", set Text color to Red (#EF4444)
    • Rule 3: If "Trend" equals "●", set Text color to Gray (#6B7280)
  • Grand Totals (Grand Audit Alignment): At the bottom of Looker Studio properties pane, check "Show summary row". This sums up columns across all Jan–Dec entries dynamically.
